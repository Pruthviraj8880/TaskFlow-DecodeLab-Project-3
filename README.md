# 🚀 TaskFlow – Interactive Productivity Dashboard

TaskFlow is a modern and responsive productivity dashboard built using **HTML, CSS, and JavaScript**.  
It helps users manage daily tasks efficiently with a premium glassmorphism UI, real-time task updates, dark/light mode, and local storage support.

---

# 🌟 Features

## Core Features:
- ✅ Add new tasks
- ✅ Mark tasks as Done
- ✅ Delete tasks
- ✅ Filter tasks (All / Pending / Completed)
- ✅ Live digital clock
- ✅ Progress bar with completion percentage
- ✅ Dynamic task statistics
- ✅ Local Storage (tasks saved after refresh)

# 📸 How It Works
1. Enter a task in the input field
2. Click **Add Task**
3. Use **Done** to complete tasks
4. Use **Delete** to remove tasks
5. Filter tasks based on status
6. Toggle between Dark/Light Mode
